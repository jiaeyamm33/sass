01_sass 함수 문법

@function 문법
사용자가 함수를 직접 정의하여 사용 가능
@mixin과 비슷해보이지만 반환 되는 내용이 다름
@mixin은 지정한 스타일을 반환하지만 @function(함수)는 연산된 값을 @return 키워드로 반환
  **문법
  @mixin 믹스인명($매개변수) {
    스타일 서식
  }
  ---------------------------

  @function 함수명($매개변수) {
    @return 값;
  } 

  **호출 문법
  @include 믹스인이름(인수);
  ---------------------------

if문법
조건에 따른 결과를 true/false로 나눠서 처리하는 삼항연산자와 비슷한 문법
조건식이 true면 참, 그렇지 않으면 false로 반환
    ** 문법
    if(조건식, 참표현식, 거짓표현식);

    조건식? 참표현식:거짓표현식;

2. @if문법

자바스크립트의 if문과 비슷하게 사용

@else를 붙여 false일 경우 실행하고, 다중 if문도 실행할 수 있다.

    **문법
    @if(조건) {
      // 참표현식
    }

    @if(조건식) {
      // 참표현식
    } @else {
      // 거짓표현식
    }

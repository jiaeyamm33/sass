1. @for문

@for 키워드는 스타일을 반복적으로 출력하는데 사용

js의 for문과 유사

  **문법
  @for $변수 from 시작 through 종료 {
    // 반복할 내용

  }

  ** 문법 2
  // to : 종료 전까지 반복
  @for $ㅂ녀수 from 시작 to 종료 {
    //반복할 내용
  }

$변수 => #{변수}

{변수}는 코드 어디든지 변수 값 사용 가능

사용 예) $img: unquote("test01.jpg")
h1 {background-color:url('./img/#{$img}')}
